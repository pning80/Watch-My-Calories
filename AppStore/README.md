# README to understand how screenshots are organized for different app versions.

The screenshots for different versions are stored in different sub-folders.

Folder name is in the format of `YYYY-MM-DD (Version X.Y.Z)`.

## Version 1.0.0

Folder `2026-03-08 (Version 1.0.0)` contains the screenshots for version 1.0.0.

## Version 1.2.0

Folder `2026-03-15 (Version 1.2.0)` contains the screenshots for version 1.2.0.
